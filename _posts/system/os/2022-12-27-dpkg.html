Debian Package Manager


Install - (dependencies)

Install

wget http://ftp.cn.debian.org/debian/pool/main/d/dpkg/dpkg_1.21.13_arm64.deb
ar x dpkg_1.21.13_arm64.deb
tar xf data.tar.xz

Config

export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/lib/aarch64-linux-gnu/:/usr/lib/aarch64-linux-gnu/
/var/lib/dpkg/status
mkdir -p /var/lib/dpkg/updates/
ldconfig
/var/lib/dpkg/info/format-new

Commands:

  • install (ignore the dependency)
    • dpkg -i --force-all xxx.deb
  • remove (ignore the dependency)
    • dpkg -r --force-all xxx
