Basic Operation

  • operate

    • add

    git add [file]

    delete

    rm -rf [files]
    git add -u

    undo uncommitted change

    git checkout -- .

    cherry pick

    git cherry-pick [commit]

    commit

    git commit -m "log"
    git push
  • status
    • git status
    git status -s
    git diff
  • log
    • git log --stat



Init

  • init
    • git init
    git remote add origin [url]

    (part of the repository)

    git config core.sparsecheckout true
    echo [relativ_path/*] >> .git/info/sparse-checkout
    git pull --depth=1 origin master

    (specific branch)

    git pull --depth=1
    git checkout --track origin/[remote_branch]



Branch

  • local

    • local branch: create and enter (based on current branch)

    git checkout -b [branch]

    local branch: create and enter (based on specified branch) (no tracking)

    git checkout -b [branch] origin/[remote_branch]

    local branch: create and enter (based on specified branch) (with tracking)

    git checkout --track origin/[remote_branch]

    local branch: delete

    git branch -d [branch]
  • remote

    • remote branch: create (no tracking)

    git push origin [local_branch]:[remote_branch]

    remote branch: create (with tracking)

    git push -u origin [remote_branch]
  • relation

    • relation: create

    git branch --set-upstream-to=origin/[remote_branch]

    relation: show

    git branch -vv
  • show

    • show local all

    git branch -a



Remote

  • operate

    • create

    git remote add [remote_name] [url]

    get info

    git fetch [remote_name]

    show

    git remote show [remote_name]

    delete

    git remote rm [remote_name]



Merge

  • check

    • check diff before merge (all)

    git diff --name-status branch_1 branch_2

    check diff before merge (all details)

    git diff branch_1 branch_2

    check diff before merge (all visual)

    git difftool branch_1 branch_2

    check diff before merge (only conflict)

    git diff branch_1...branch_2

    check diff before merge (one)

    git diff branch_1...branch_2 -- one_file
  • operate

    • merge

    git merge [branch]

    merge without commit

    git merge --no-commit [branch]
  • graphic tool
    • git mergetool
  • upload
    • git push



Tag

  • operate

    • create (lightweight)

    git tag [name]

    create (annotated)

    git tag -a [name] -m 'log'

    show

    git show [name]

    upload

    git push origin [name]

    upload all

    git push origin --tags
