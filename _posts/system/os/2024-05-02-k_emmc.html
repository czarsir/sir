Specification

embedded MultiMediaCard (eMMC)

Mode

transfer mode frequency bits max bandwidth
DS SDR 26 MHz - 26 MB/s
HS SDR/DDR 52 MHz - 104 MB/s
HS200 SDR 200 MHz - 200 MB/s
HS400 DDR 200 MHz - 400 MB/s
DS: Default Speed
HS: High Speed
SDR: Single Data Rate
DDR: Dual Data Rate

Regs

  • CID : id
  • RCA : relative device address
  • DSR :
  • CSD :
  • OCR :
  • EXT_CSD :


Hardware

  • Host -> eMMC : clock
  • Host <- eMMC : data strobe (with 5.0 version)
  • Host <-> eMMC : cmd
  • Host <-> eMMC : data[0-7]
  • (Following can utilize GPIO)
  • Host -> eMMC : reset


Workflow

  1. reset host
  2. reset emmc
  3. cmd 0 : reset
  4. cmd 2 : get CID
  5. cmd 3 : set RCA
  6. cmd 8 : get EXT_CSD via data lines
  7. cmd 6 : switch operation mode
