Image

  • load

    • (from remote)

    podman pull docker.io/library/httpd

    (from local)

    podman load -i demo.tar
  • show
    • podman images



Container

  • create

    • interactive

    podman run -it demo:[demo_tag] /bin/bash

    background

    podman run -dt demo:[demo_tag] /bin/bash

    [option] mount

    podman run -it --mount type=bind,src=/host_dir,dst=/container_dir[,U=true] demo:[demo_tag] /bin/bash

    [option] volume (bind mount)

    podman run -it -v /host_dir:/container_dir demo:[demo_tag] /bin/bash

    [option] multi volume

    podman run -it -v /host_dir1:/container_dir1 -v /host_dir2:/container_dir2 demo:[demo_tag] /bin/bash
  • enter a specific container
    • podman start [demo_id]
    podman attach [demo_id]
  • delete
    • podman rm [demo_id]
  • show all
    • podman ps -a
